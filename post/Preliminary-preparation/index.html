前期准备

2019-12-12 2 min read # 电商系统开发

对于有软件研发能力的商家，目前主流的电商平台的开放平台，均提供 API 接口供开发人员调用，可以方便的处理订单、自动发货、同步库存等操作。下面以入驻天猫聚石塔为列，介绍商家后台管理系统开发的前期准备工作。

创建应用

点击聚石塔，点击左上角的登录链接，页面跳转至登录页，输入天猫商家主账号和密码，点击登陆按钮，返回聚石塔主站，如下图。

点击控制台链接，进入聚石塔的控制台主页，如下图：

点击应用管理，进入应用管理主页，如下图：

点击 ➕ 号，创建应用。

选择应用类型为 电商后台 ，如下图：

选择应用类型为 我是天猫/集市商家，点击创建应用，如下图：

选择应用类型为 商家后台系统，点击申请申请，跳转页面到填写应用信息和开发者信息页面，填写相关信息，完成应用创建。

点击应用管理，进入应用 APP 证书，如下图：

点击上图的应用管理，可以查询到 App Key 和 App Secret。这两个 APP 参数对于 API 对接，是最重要的两个参数。

点击 SDK 下载，可以下载对应版本的 SDK，来完成 API 接口的快速调用和接入。如下图：

点击安全中心下的 IP 白名单设置，填入经你允许的 IP 地址，为方便调试，可以添加你本地的 IP。如下图：

注意事项

  1. 提供的产品说明书 聚石塔官方提供产品说明书模板，按要求填写即可；
  2. 需提供商家自研软件的著作权证书，关键是第二点，现在审核商家自研应用非常严格，必须提供软件著作权证书，才能通过应用的审核。
